# Memory Link

A small browser puzzle game about opening forgotten portrait-compartments, recovering memories, and reconnecting with an old friend.

## Play

Open `index.html` in a browser.

The game is self-contained: no build step, install step, or external assets are required.

## GitHub Pages

To publish with GitHub Pages:

1. Create a new GitHub repository.
2. Add `index.html` and this `README.md` to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Set the source to your main branch and root folder.
5. Save, then wait for GitHub Pages to publish the site.

## Files

- `index.html` - the complete playable game
- `README.md` - project notes and publishing instructions
